HowTo Request Removal of User Data

  • Login to the application (login Icon in the top right corner)
  • Navigate to my-profile ("/my-profile" or click on the user icon in the top left corner)
  • Click the Button "Request User Data Removal" and confirm the action with "Yes"
  • The Admin team will be notified about your request immediately and remove / anonymize your data in the system
  • You will receive a confirmation of the deletion request by Mail
